# Netflix-Data
This project analyzes Netflix’s global content catalog to uncover trends in genres, countries, and release years. Using cleaning, preprocessing, visualizations, and a simple predictive model, it provides insights into how Netflix distributes and expands its worldwide entertainment offerings.

# Dataset Description
The dataset contains 6,000+ rows × 12 columns with the following fields:
## Columns
* show_id – Unique identifier for each Netflix title
* type – Content category: Movie or TV Show
* title – Name of the movie or series
* director – Director(s) of the production
* cast – Actors featured in the title
* country – Country or countries of production
* date_added – Date the title was added to Netflix
* release_year – Original release year of the content
* rating – Maturity rating (TV-MA, TV-14, PG, etc.)
* duration – Runtime of movie or number of seasons for TV shows
* listed_in – Genres/categories assigned to the title
* description – Short summary or storyline description

# Key Insights (Summary)
* Netflix has significantly grown its catalog since 2015.
* Movies make up the majority of titles.
* The US and India are top contributors.
* Drama, International content, and Documentaries dominate genres.
* Most content falls under adult-oriented ratings (TV-MA).
* Movie duration averages 90–110 minutes.
* TV Shows generally have 1–2 seasons.
* January, March, and October see the most new additions.
* ML model shows content duration cannot be predicted well from metadata.

# Conclusion
This project provides a detailed exploration of the Netflix dataset through cleaning, preprocessing, visualization, and predictive modeling. The analysis reveals how Netflix’s library
has evolved, highlights major trends in genres and countries, and provides insights into user-consumption patterns. While simple ML models show limited predictive power, the study
demonstrates strong potential for richer models using text and advanced features.
